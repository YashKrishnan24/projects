🎨 Project 1: Glassmorphism Email Submission
A sleek, mobile-first UI focused on modern transparency and fluid UX.

This project features a high-end Email Submission Page using the Glassmorphism design trend. Built with pure HTML/CSS and Vanilla JS, it emphasizes soft aesthetics without sacrificing performance.

✨ Key Features
Glassmorphism UI: Leverages backdrop-filter: blur() and subtle gradients for a premium, frosted-glass look.

Mobile-First Design: Fully responsive layout that scales beautifully from smartphones to desktops.

Floating Labels: Interactive input fields that provide a smooth, modern micro-interaction.

Zero-Dependency: Lightweight and fast, using no external frameworks or libraries.

Smart Validation: Built-in client-side email verification for a seamless user experience.

☄️ Project 2: The Great Gravity Glitch (Java)
A fast-paced, neon-style arcade challenge built in Java Swing.

The Great Gravity Glitch is a reflex-based game where the player battles "glitching" gravity. Unlike traditional catch games, the user defines the stakes by setting their own target before the 20-second chaos begins.

🕹️ Key Features
Custom Goal Setting: You choose your difficulty. Set a target catch count and race against the clock to hit it.

Dynamic Difficulty: As the timer bleeds out, the gravity "glitches," causing balls to drop at increasing speeds.

Neon Aesthetics: Features a high-contrast synth-wave visual style with a deep-space background.

Reflex Testing: Constant basket size and randomized spawn rates ensure every round is a test of pure skill.

Win/Loss Logic: Real-time scoring with instant feedback—reach your goal to save the universe or face the "Hard Luck" screen.
